### 1. bank_account.py
A banking system with account classes.

**How to run:**
```
python3 bank_account.py
```

**What it does:**

- `SavingsAccount` class: Inherits from BankAccount with minimum balance requirement
- Includes functions for deposit, withdraw and display balance

### 2. decorators.py
Simple timer decorator example.

**How to run:**
```
python3 decorators.py
```

**What it does:**
- Times how long the `testing()` function runs
- Uses a decorator to add timing functionality
- Sleeps for 3 seconds as demonstration

**How to run:**
